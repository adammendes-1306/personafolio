# PersonaFolio

A modern and responsive portfolio website built to showcase my skills, projects, and experience as an aspiring Software Developer.

## Live Demo

https://adammendes-1306.github.io/personafolio/

## Features

- Responsive portfolio website
- Dynamic content powered by Firebase Firestore
- Secure admin panel using Firebase Authentication
- Project showcase
- Skills management
- Experience timeline
- Contact information

## Built With

- HTML5
- CSS
- JavaScript
- Firebase Authentication
- Cloud Firestore
- GitHub Pages

## Getting Started

1. Clone this repository.
2. Open the project in Visual Studio Code.
3. Start the project using Live Server.
4. Create your own Firebase project and update the Firebase configuration.
5. Enable Firebase Authentication (Email/Password) and Cloud Firestore.
6. Access the admin panel at `/admin/login.html`.
7. Sign in using an authorized Firebase Authentication account to manage portfolio content.
